Example CSS animation idea

If your logo is built as SVG with IDs like bar1, bar2, bar3, and iris-ring, this works:

CSS
#bar1, #bar2, #bar3 {
  transform-box: fill-box;
  transform-origin: bottom center;
  animation: barPulse 1.8s ease-in-out infinite;
}

#bar2 {
  animation-delay: 0.15s;
}

#bar3 {
  animation-delay: 0.3s;
}

#iris-ring {
  transform-box: fill-box;
  transform-origin: center;
  animation: rotateEye 8s linear infinite;
}

@keyframes barPulse {
  0%, 100% {
    transform: scaleY(1);
  }
  50% {
    transform: scaleY(1.18);
  }
}

@keyframes rotateEye {
  from {
    transform: rotate(0deg);
  }
  to {
    transform: rotate(360deg);
  }
}
